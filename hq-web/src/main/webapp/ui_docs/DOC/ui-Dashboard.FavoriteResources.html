About the Favorite Resources Portlet

The Favorite Resources portlet lists the name, resource type, current availability, and total fired alerts for selected resources.

Select a Favorite Resource On the Fly

You can add a resource to Favorite Resources when you are viewing it in the Resources tab — click Add to Dashboard Favorites on the Tools menu.

Configure Favorite Resources on the Dashboard

  1. Click the gear icon in the upper right corner of the Favorite Resources portlet.
  2. The HQ Dashboard Settings: Favorite Resources page appears.
  3. If the Selected Resources section contains resources that you do not want to be included in the portlet, checkmark them and click Remove From List.
  4. To add resources to the portlet:
    1. Click Add to List.
    2. The Dashboard Settings: Favorite Resources: Add Resources page appears.
    3. In the Resources column:
      1. Use the View filters to filter by inventory type, resource type, or both, as desired.
      2. Enter a substring in the Filter by Name field to filter by resource name, as desired.
      3. In the list of resources, checkmark desired resources, and click the right arrow between the columns to include them in the portlet. 
    4. If the Add Resources column contains resources you do not wish to include in the portlet, checkmark those resources and click the left arrow between the panels
  5. Click OK.

I've beefed up the description of the Favorite Resources portlet on the help page for the Dashboard: http://support.hyperic.com/confluence/display/DOC/ui-Dashboard

For your immediate edification (I hope): What are performance metrics? Explained here: http://support.hyperic.com/confluence/display/DOC/Managing+Metric+Data#ManagingMetricData-Performance.

What is shown in the Performance column? If a single performance metric is being collected for the resource, that value is displayed. If several performance metrics are being collected, then HQ chooses the one marked as an indicator metric. An "N/A" value indicates that no performance metrics are being collected for the resource. To see what performance metrics are being collected for a resource, go to the Metrics Data tab in the "Current Health" screen for the resource.

As for the "proper" configuration of the portlet, there really isn't any such thing. All you do as a user is choose resources that are your "favorites" (either directly in the portlet or via the Tools Menu when viewing a specific resource) and HQ puts them all in this portlet with the data explained in the help documentation. Another way to say that is that the portlet is always properly configured.
