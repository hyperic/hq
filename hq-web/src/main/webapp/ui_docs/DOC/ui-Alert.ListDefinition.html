The Configure view of the Alert tab for a resource lists all alert definitions for the resource, and allows you to create, edit, enable/disable, and delete alert definitions for the resource.

The default view when you click the Alert tab for a resource is a list fired alerts.  Click Configure to list alert definitions. 

Notes for HQ Enterprise users

The information on this page applies to resource alert definitions. To view and work with resource type alert definitions, go to the alert definition list for the target resource type.

  1. Select the Administration tab.
  2. Click Monitoring Defaults in the "HQ Server Settings section".
  3. Click Edit Alerts in the row for the target resource type.

List Alert Definitions for a Resource

The following information is shown for each alert definition that is assigned to the resource, or in HQ Enterprise, inherited by the resource from a resource type alert definition.  An inherited alert definition has a red asterisk preceding its name in the "Alert Definition" column.

Label
Description
Alert Definition
Name assigned to the alert definition.
Description Description defined in the alert definition.
Date Created
Date and time the alert definition was created.
Last Modified
Date and time the alert definition was most recently modified.
Active Indicates whether the alert definition is currently enabled or disabled. "Yes" and a yellow flag in this column indicates that the alert definition has fired once and is disabled until it it is "Fixed". (The alert definition was configured to "fire once and disable until fixed".) When the alert is fixed the definition will be re-enabled.

Create an Alert Definition for a Resource

Click and proceed to the "New Alert Definition" page - the new alert definition will apply to the currently selected resource. For more information, see the associated help page.

Enable or Disable a Resource Alert Definition

When an alert definition is "enabled" it is active and alerts can fire. When an alert definition is "disabled", alerts for it will not fire.

Check the alert definition name, and in Set Active, select "YES" (to enable) or "NO" (to disable) in the dropdown box and click .

Note: If you use HQ Enterprise, do not disable an alert definition marked with an asterisk (*) - this indicates it is inherited from a resource type alert definition.  Changes made to an inherited alert definition from a resource's Alert tab will be overridden by subsequent updates to the resource type alert definition. To enable or disable a resource type alert definition, go to the list of alert definitions for the resource type, as described in the note at the top of this page.

Edit a Resource Alert Definition

Check the alert definition name and proceed to the "Edit Alert Definition" screen.

Note: If you use HQ Enterprise, do not use the edit option on this page for an alert definition marked with an asterisk (*) - this indicates it is inherited from a resource type alert definition.  Changes made to an inherited alert definition from a resource's Alerttab will be overridden by subsequent updates to the resource type alert definition. To edit a resource type alert definition, go to the list of alert definitions for the resource type, as described in the note at the top of this page.
