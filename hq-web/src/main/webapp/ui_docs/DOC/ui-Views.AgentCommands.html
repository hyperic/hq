Agent Control Commands*

Available only in HQ Enterprise

This page, which you can select from the Views tab for an HQ Agent, or a group of agents, in inventory, allows you to perform selected operations on the agent.

Note:  Agent control commands are only available for HQ Enteprise Agents version 4.0 and later. 

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on this Page

Restart the agent

This command invokes the agent's Java Service Wrapper's restart command. The restart command shuts down the JVM process in which the agent runs, waits for the process to terminate cleanly, and spawns a new JVM process for the agent. During the restart process, the agent's metric collection and resource control functionality will be interrupted.

The restart command happens asynchronously. To verify that the restart succeeded you can go to the page for the agent in the HQ Portal and check its availability. Alternatively, you could configure an alert for the agent that fires when the agent's availability changes. 

To restart an agent: 

  1. Select restart from the pull-down list
  2. Click Execute. 

Ping the agent

  1. Select ping from the pull-down list
  2. Click Execute.

Upgrade an agent to a newer version

When you invoke this operation:

  • the selected agent bundle is transferred from the HQ Server to the target agent(s)
  • the agent expands the bundle locally
  • the agent updates the local bundle property
  • the server restarts the agent

The configuration settings in the agents /conf file are preserved.
To initiate an agent upgrade:

  1. Select upgrade from the agent operation pull-down list.
  2. Select a bundle from the agent bundle pull-down list.
  3. Click Execute.

Push a plugin to the agent

This operation allows you to transfer new, customer, updated, or patched plugins from the HQ Server to the target agent(s).  Pushing plugins to an agent results in an agent restart.

  1. Select push plugin from the agent operation pull-down list.
  2. Select a plugin from the plugin pull-down list.
  3. Click Execute.
