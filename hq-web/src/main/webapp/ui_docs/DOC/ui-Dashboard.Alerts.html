This screen allows users to control which alerts are displayed in the "Recent Alerts" portlet on the Dashboard.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To control which alerts are displayed in the portlet:

  1. (optional) Type the portlet's Description. This description shows up at the top of the portlet
  2. In Alert Range, select the set of alerts to be displayed in the portlet: select how many recently triggered alerts to show, the minimum priority the alert must have, and the time frame during which the alert must have been triggered.
  3. Also in Alert Range, select whether HQ should report alerts for all managed resources ("all resources") or for only those listed lower in the screen ("selected resources").
  4. If the alerts are restricted to "selected resources," specify those in "Selected Resources."
    • To add a resource, click and proceed to the "Dashboard Settings: Add Resources" screen.
    • To remove a resource, check the resource and click .
