HQ Dashboard Settings: Problem Resources

Available only in HQ Enterprise

Pre-Release Documentation
This space is a staging area for Hyperic 4.5 documentation. See HQ Documentation for Hyperic 4.2, 4.3, and 4.4 documentation.

This screen allows users to change the display of resources in the "Problem Resources" portlet on the Dashboard. The screen lists all the problem resources (resources with problem metrics), their availability, the number of alerts triggered on each of them, and their OOB metric values.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To change the portlet display:

  1. Specify the following values:
    • Show Maximum of: The maximum number of problem resources to be displayed in the portlet
    • For the Last: The time range to look in when finding problem resources
  2. Click

To view metrics for a problem resource:

  • Click the resource name and proceed to the "Current Health" screen for the resource.
