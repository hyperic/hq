vFabric Hyperic 4.6.5 : Starting the Server on Unix-Based Platforms
This page last changed on Jan 04, 2012 by mmcgarry.

Start the server with this command: 

<Server Installation Directory>/bin/hq-server.sh start

The script will display some startup information on stdout, then it will detach and run in the background

The server.log file and and bootstrap.log files in the logs directory under the server installation directory will have detailed startup information.
