Editing a Server's Install Path

Topics marked with * relate to HQ Enterprise-only features.

This screen allows users to edit a server's install path. The server's server type and the platform that hosts the server are also listed.

Feedback is welcome. Click Add Comment at the bottom of the page.

Tasks Available on This Screen

On this screen, users can do one thing: edit a server's install path.

To edit a server's install path:

    Use the full path of the directory where the server software is installed.
