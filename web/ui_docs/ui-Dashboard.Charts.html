HQ Dashboard Settings: Saved Queries

From this page you can customize which saved charts are displayed on the Dashboard.

To toggle the entire Saved Charts portlet display on or off, select the YES or NO radio button.

Choose what saved charts you want displayed by checking the appropriate checkbox in the Selected Charts list. If you have not saved any charts to the Dashboard, no charts will be listed here. To remove a chart from the list, select its checkbox and click Delete. To save your changes and return to the Dashboard, click OK.
