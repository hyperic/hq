Browse Resources

From the Browse Resources page, you can locate and navigate information about all the resources Hyperic HQ is managing.

When you view resources in the Browse Resources page, they are displayed by inventory type (Platforms, Servers, Services, Compatible Groups/Clusters, Mixed Groups, or Applications). The default type is Platforms. To view resources of a different type, click on the type you want to view from the Resource Bar. The Resource Bar displays all types currently in inventory and the total count of each type.

You can filter the list of resources by the resource type by selecting the desired type from the View drop-down. The table will then redisplay those resources and their indicator metric values.

To change the set of indicators, go to the Administration link up in the masthead. From there click on Monitoring Defaults Configuration. On that page click the 'Edit Metric Template' link next to the resource type you want to change. (i.e. Linux). The resulting page allows you to select different metrics as indicators, as well as change the collection interval. Note that changes in this area will affect all existing resources of that type in your inventory.

By default the resources are listed using a textual output. To change to a graphical view, click on the CHART VIEW button on the upper right hand corner.

When using the text listing, you can navigate directly to it's inventory, monitoring, or control pages directly by using the Quick Nav icons.

Browse Resources Quick Nav icons

Icon Action
Clicking will take you to the resource monitor page.
Clicking will take you to the resource inventory page.
Clicking will take you to the resource control page.
Indicates this feature is not available due to permissions.

To aid in finding the resource you are looking for, you can search for resources matching a query using the Keyword Search . You can also limit the number of resources listed on the page by using the Items Per Page drop-down on the bottom of the listing. You can also sort the items by clicking on the column heading.
