Availability Summary Add/Remove Resources

This screen allows users to add and remove resources from the Dashboard's "Availability Summary" portlet.

Tasks Available on This Screen

On this screen, users can do the following tasks:

To add a new resource to the portlet:

  1. Check the desired resources in "Resources" and click the right arrow button between the two lists to move the selected resources to "Add Resources." (The arrow is enabled only when some resources are selected.) All managed resources are available for selection.
    Users can filter the list of available resources either by:
    • Selecting an inventory level and a specific resource type in View. (Platforms are grouped by OS type.) or
    • Typing (part of) the resource's name in Filter By Name and clicking the arrow button directly to the right.
  2. Click OK.

To remove a resource from the portlet:

  1. Check the desired resources in "Add Resources" and click the left arrow button between the two lists to move the selected resources to "Resources." (The arrow is enabled only when some resources are selected.)
  2. Click OK.
