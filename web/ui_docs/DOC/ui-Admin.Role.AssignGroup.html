Assigning Groups to a Role

Available only in HQ Enterprise

This screen allows users to assign existing existing groups to a role. Assigning a group to a role gives the role's users access to the group's component resources. Without groups, a role has no resources and therefore its users cannot see any resources.

If you have any comments or suggestions for this help page, please submit them at the bottom of the page by clicking Add Comment.

Tasks Available on This Screen

On this screen, users can do one thing: assign groups to a role.

To assign a group to a role:

  1. Check the desired groups in "Groups" and click between the two lists to move the selected roles to "Assign To Roles." (The arrow is enabled only when at least one group is selected.)
  2. Click .
    The role's users can now access the group's resources.
