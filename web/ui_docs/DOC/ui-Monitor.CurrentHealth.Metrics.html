View Metrics

This page presents all of the metric data that has been collected by HQ in the metric display range in aggregate form. However, the view can switch between displaying all metrics and those with data. To see all metrics, click on Show All Metrics: . To switch back to only metrics with data, click on Hide Metrics Without Data: , just below the tab. The metrics are displayed metrics with their low, average, peak, last values, and collection interval, as well as alerts and out-of-bounds counts, for the specified metric display time range. You can chart or set baselines of one or more of the metrics from this list. In the column of this page, there are options to filter the complete list by metric categories, collection type, or keyword. The Resources tab presents the resources in the hiearchy of the current resource being viewed. In addition, the list of metrics can be filtered by their categories, value types, or keyword.

Modifying Metrics

When you have added a resource to the Resource Hub and set its Configuration Properties, you can specify what metric data you want Hyperic HQ to collect and display for that resource. The metrics that are available for a particular resource will depend on that resource. You can see the list of available metrics in the Plugin Reference.

To modify the metrics being collected for a resource, first navigate to view the metric data of that resource.

To Add Or Edit Metric Collection

The metrics that are not currently being collected by Hyperic HQ are listed with Collection Interval of NONE. The metrics that are currently being collected by Hyperic HQ are listed with their specified Collection Intervals. To enable or modify metric collection intervals, select the metrics you want to modify by checking the checkbox next to the metric name(s). Next, enter the collection interval value you want in the Collection Interval for Selected field, selecting a time unit from the drop-down list next to the field. Click the button to update the metrics with the collection interval you specified.

Working With Autogroup Metrics

For an autogroup, Hyperic HQ lists all metrics available for collection for that resource type. The Members Collecting column of each metric denotes if any of the members of the autogroup is collecting that metric. The possible values are NO (none), YES (all), or SOME. If Hyperic HQ is currently collecting a given metric, it is listed with its specified Collection Interval, if all members that are collecting the metric have the same collection interval; otherwise, the column displays DIFFERENT. To enable or update the collection interval for metrics, check the checkbox next to their names and enter the value you want in the Collection Interval for Selected field, selecting a time unit from the drop-down list next to the field. Click the button to update the collection interval for that metric for all members of the autogroup with your new value.

To Stop Metric Collection

To discontinue data collection for metrics which are currently being collected by Hyperic HQ, check the checkbox next to their names and click the button to disable them.
